Data Immersion & Wrangling

Project Overview

This project focuses on understanding, cleaning, and preparing the titanic dataset for analysis. The goal was to identify data quality issues, perform data cleaning , and create an analysis-ready dataset.

Dataset Used
Titanic Dataset

Objectives
--> Understand the dataset structure
--> Identify missing values
--> Detect duplicate values
--> Clean and transform records
--> Create useful features
--> Prepare a clean dataset for analysis

Tools Used
--> Python
--> Pandas
--> VS Code
--> Microsoft Excel
--> Git & GitHub

Data Cleaning Steps Performed
1. Loaded the dataset using pandas.
2. Explored the dataset structure.
3. Identified the missing values.
4. Filled missing Age values using the mean.
5. Filled missing Embarked values using the mode.
6. Removed the Cabin column due to excessive missing values.
7. Removed the duplicate records.
8. Created a new feature called FamilySize.

Output 

A cleaned and transformed Titanic dataset ready for further analysis and visualization.
